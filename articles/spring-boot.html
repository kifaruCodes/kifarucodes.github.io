Spring Boot

Overview

Spring Boot is an open-source Java framework based off of Spring. It is used to build Java products in a way that makes it easy to take them to production

Spring boot therefore abstracts the nitty gritties of development and deployment so that one focused on the product rather than the technologies.

Compared to Spring, where configuration is done explicitly, Spring boot has features such as automatic configuration of dependencies (auto-configuration), as well as production features (actuators), hence making development and production much easier.

In this documentation article, I go through different Spring Boot functionalities that I use to develop products and solutions.
