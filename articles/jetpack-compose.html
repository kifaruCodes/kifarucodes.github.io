Jetpack Compose

So what is Jetpack Compose? This is an Android toolkit that makes it easier to develop Android User Interface. Before Jetpack, we had Android UI developed using XML. One of the reasons many prefer using Jetpack is due to the fact that developing Android applications is all in one languate, as Jetpack is coded in Kotlin. This therefore means that there is no need to learn a new language i.e. XML.
